# woocommerce-customer-shop-access
This is a WordPress plugin that allow the admin to grant certain users access to the WooCommerce shop.

## Features

- Individual customer approval
- WooCommerce shop protection
- Navigation menu integration
- User profile approval checkbox
- AJAX approval
- Bulk approval
- Settings page

## Requirements

- WordPress 6.8+
- WooCommerce 9+
- PHP 8.1+
